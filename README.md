# 十九路围棋 · Android

一个面向手机端的离线围棋人机对战游戏。棋盘采用标准 19×19 规格，玩家可以自由选择执黑或执白，并使用双方独立的正式比赛计时。

## 功能

- 标准 19×19 棋盘与九个星位
- 玩家自由选择执黑先行或执白后行
- 提子、禁自杀、简单劫争
- 双方连续停一手后终局数子
- 白棋贴 7.5 目
- 悔棋、重新开局、落子提示
- 双方独立计时，支持主时间与多次读秒
- 三种计时预设：
  - 快棋：10 分钟主时间＋3 次 30 秒读秒
  - 标准赛：30 分钟主时间＋5 次 60 秒读秒
  - 长考：60 分钟主时间＋5 次 60 秒读秒
- 普通、强势、碾压三档本地 AI
- 完全离线运行，无广告、无联网权限

## 自适应 AI

AI 不再使用固定搜索次数。每回合会综合以下因素动态决定搜索预算与模拟深度：

- 当前处于布局、中盘还是官子阶段
- 合法候选点数量与局面分支宽度
- 打吃、提子等战术紧迫程度
- 棋盘上的棋块数量与复杂度
- AI 自己的主时间、读秒次数和当前读秒余量

碾压难度的单回合搜索上限为 **3000 步**。时间充足且中盘战斗复杂时会接近上限；布局简单、官子明确或临近超时时会主动缩短搜索，避免机械地浪费时间。

> 本项目使用启发式候选剪枝和蒙特卡洛推演，不是 AlphaGo 神经网络，也无法达到专业围棋引擎的棋力。

## 版本

当前 Android 版本：`1.1.0`（versionCode 2）

## 获取 APK

打开仓库的 **Actions** 页面，进入最新一次 **Build Android APK** 工作流，在页面底部下载 `Wuziqi-Go-debug-apk` 构建产物并解压，即可得到 `app-debug.apk`。

也可以进入 Actions 手动运行 `Build Android APK`。

## 本地构建

需要 JDK 17、Android SDK 34 与 Gradle 8.7：

```bash
gradle :app:assembleDebug
```

APK 输出位置：

```text
app/build/outputs/apk/debug/app-debug.apk
```
